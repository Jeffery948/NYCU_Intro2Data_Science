Deal with Missing Value

Data Visualization

1. Type of Content

Bar Plot



  • 我用 bar plot 來分析 movie 和 Tv show 的數量, 並用 text 函式將數量顯示在圖上, 我會選擇 bar plot 的原因是因為非常直觀

2. Release Year Analysis

Box Plot (with Outliers)



Box Plot (without Outliers)



  • 我會使用 box plot 的原因是整份資料只有 release year 是 numeric, 故可以輕易地取得四分位數. 而對比兩張圖可以發現對 release year 來說, 有沒有 outliers 的存在差非常多, 可以得知原始資料的極端值不少, 因此影響頗大

Density Plot



  • 我使用了 ggplot2 的 geom_density 來觀察 release year 的分布, 可以看到大部分都分布在近年, 剩下在早年的皆為 outliers, 和上面的結果吻合

3. Country-wise Analysis

Pie Chart



  • 我使用 pie chart 來觀察前十產量的國家, 原因是效果會比 bar plot 來的好, 結果清晰易見

4. Date Added Analysis

Time Series Plot



  • 我使用 time series plot 來分析 date added, 因為我覺得非常適合這份資料, 可以明顯的看出哪些年有比較多的作品加入, 而因為此資料原本是 character, 故我先把它轉成 date format, 再把年份取出來才完成此 plot

5. Rating Analysis

Pie Chart



  • 我會使用 pie chart 來分析 rating 的原因依舊是我發現效果比 bar plot 好, 而我在畫圖的過程中發現有太多不同的 rating, 導致整個圖非常亂, 因此我把數量小於四百的 rating 都放到 other 裡, 圖片變得整齊許多

6. Correlation Analysis

Scatter Plot



  • 我用 scatter plot 來觀察 duration 和 release year 之間的關係, 可以發現隨著時間過去, 長度較短的電影越來越多, 而其他時間的電影也有變多, 但沒那麼明顯. 由於 duration 中有一些是 season, 所以我還要先把 season 都剔除並且相應的 release 也要解決, 才完成此 plot

7. Simpson’s Paradox Analysis



  • 這張圖是 Tv show 和 release year 的關係圖, 我把一季當作十集且一集三十分鐘由此來分析, 可以發現隨著時間過去, 長度較長的 Tv show 逐年增加




  • 這張圖是 movie 和 Tv show 的綜合圖(我把 druation 限制在 1500, 比較方便觀察 movie), 接下來我要分別找到 movie 和 Tv show 的回歸曲線, 並將兩者合併觀察




  • 畫完回歸線的圖長這樣, 藍色線是代表 Tv show, 紅色線是 movie, 而綠色線則是兩者合併得來, 可以發現藍色有很明顯的下降趨勢, 而紅色雖然不明顯但也有在下降, 結果兩者合併後的綠色居然有明顯上升的趨勢, 這證明了我的資料集有 Simpson’s paradox 的存在

Conclusion

  • 我的資料大部分都是 character, 故很難進行 scatter plot, 只能盡可能找最合適的資料進行轉換再畫圖

  • 我在檢查是否有 Simpson’s paradox 時把 1 season 直接換算成 3000 minutes, 實際上的情況可能會有誤差, 但我認為 Tv show 的回歸線依舊會是下降趨勢, 故我的資料集應該是真的有 Simpson’s paradox

  • 我認為把資料視覺化確實對理解資料有幫助, 閱讀的時候可以更加快速, 也更容易理解

Possible Problems

  • 這次還有許多畫圖插件沒用到, 例如: histogram, QQplot… 等, 因為我的資料型態不適合使用這些, 希望以後有機會使用到

  • 我找 Simpson’s paradox 只用了一個例子, 說不定我的資料中還有其他的 Simpson’s paradox, 我可以嘗試尋找其他的

  • 我對資料型態間的轉換還是不太熟悉, 我想要早點熟練運用, 畢竟我的資料型態實在是不適合數值分析
