LDA (Linear Discriminant Analysis) :-is a dimensionality reduction technique.

-Linear Discriminant Analysis (LDA) is a supervised learning algorithm used for classification tasks in machine learning. It is a technique used to find a linear combination of features that best separates the classes in a dataset.

-LDA or Linear Discriminant Analysis can be computed in R using the lda() function of the package MASS. -The prior:- argument sets the prior probabilities of class membership. If unspecified, the class proportions for the training set are used.

Prior probabilities of group: These represent the proportions of each region in the training set.For example, 11.11% of all observations in the training set were of region Apulia.north.

Group means: These display the mean values for each predictor variable for each region.

Coefficients of linear discriminants: These display the linear combination of predictor variables that are used to form the decision rule of the LDA model.For example LD1: 0.004779161palmitic + 0.006786701palmitoleic + 0.015331122*stearic …..

Proportion of trace: These display the percentage separation achieved by each linear discriminant function.

Quadratic Discriminant Analysis (QDA)

-QDA is generally preferred to LDA in the following situations:

  1. The training set is large.

  2. It’s unlikely that the K classes share a common covariance matrix.

-QDA is implemented in R using the qda() function, which is also part of the MASS library.
