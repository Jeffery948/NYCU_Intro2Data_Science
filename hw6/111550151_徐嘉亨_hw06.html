Regression Models

Simple Linear Regression



  • 我先用 corrplot 畫出關係圖, 發現和 quality 最有關聯的是 alcohol (0.48), 因此我決定選擇 alcohol 當 variable




  • 我使用 lm function 做出 quality 和 alcohol 的迴歸線, 並用 summary 分析結果


  • 用 coef 分析 linear regression 的係數, 可以得知 quality = 1.8749479 + 0.3608418 * alcohol




  • 用 confint 分析 confidence interval of the estimated parameter




  • 我把 quality 和 alcohol 的 scatter plot 和迴歸線放在一起, 可以發現兩者之間是正相關




  • 用 residuals 來觀察實際的 quality 值和估計的 quality 值之間的差距, 由於篇幅過長, 只擷取一小部分




  • 我用 QQplot 畫出了 normality of the residuals from a linear regression model




  • 我使用 predict 來預測新資料的 alcohol 對應的 quality

Polynomial regression



  • 用 lm function 求出 quality 對 alcohol 的二次式, 並用 summary 分析結果




  • 用 ggplot 畫出剛剛結果的圖, 可以發現對應到的 quality 幾乎都是整數




  • 用 lm function 求出 quality 對 alcohol 的三次式, 並用 summary 分析結果




  • 用 ggplot 畫出剛剛結果的圖, 可以發現對應到的 quality 一樣幾乎都是整數




  • 觀察各項的誤差平方




  • 分別用二次式和三次式來預測 quality

1. Backward Elimination



  • 可以發現 citric.acid 被去除掉了




  • 用 summary 觀察結果

2. Forward Selection



  • 可以發現 alcohol, volatile.acidity, and sulphates 在 steps 中被加進來




  • 用 summary 觀察結果, 結果和 backward elimination 一樣

3. Stepwise Regression



  • 可以發現 citric.acid 被去除掉了




  • 用 summary 觀察結果, 結果和前兩個一樣

Analysis of Variance (ANOVA)

  • The result shows a Df of 1, and a very small p-value (< .001). This means that adding the sulphates to the model 2 did lead to a significantly improved fit over the model 2

Checking for the Collinearity



  • 觀察四個變數的 Variance Inflation Factor, 可以發現沒有一個超過 10, 故沒有 problematic collinearity

Locally Weighted Regression

Kernel Regression



  • 用 ksmooth 求得 kernel regression, 並畫成曲線和原數據對比

Conclusion

  • 根據觀察發現, 我使用 parametric regression(Simple Linear Regression, Polynomial regression, Multiple Linear Regression) 來預測 quality, 預測的結果都差不多, 不過和實際情況差的不少, 我推測是因為 quality 的範圍太小, 只有 3~8, 而且又是整數型態, 造成如此結果

  • 而我使用 non-parametric regression(Locally Weighted Regression, Kernel Regression) 來預測 free.sulfur.dioxide, 結果不太一樣, 和實際數據也沒有很接近, 我推測是因為 free.sulfur.dioxide 和 total.sulfur.dioxide 所形成的點太密集, 無法有效分析
